# Statistics — Early Work (2019)

Notebooks from my MSc in Statistics at University of Malaya. Written to understand the math behind the models, not just use the API.

## Notebooks

### Linear Regression — From First Principles
Builds linear regression step by step from variance and least squares, through R², F-statistics, residuals, and QQ plots. Uses `scipy`, `sklearn`, and `matplotlib`.

### Bayesian Linear Regression (PyMC3)
Bayesian approach to the same problem — priors, posteriors, MCMC sampling with PyMC3. Compares frequentist vs Bayesian results on the same dataset.
